What is angular?

According to Angular's official documentation, Angular is a component-based framework for building scalable web applications, also it is a collection of well-integrated libraries that cover a wide variety of features, including routing, forms management, client-server communication, and more.

For this framework, the essentials are: components, templates and dependency injection. Components are basically building blocks that together form the app. For example, this "About" section is a single component, the next section will be another component and so on. There is not only a single way of defining components in your project, these are defined depending on what is best for the development of the project.

Templates are basically HTML code that exchanges information with Angular in order to render the code in the browser. And dependency injection is to request anything from an outside class rather than to create a resource inside that same class.
